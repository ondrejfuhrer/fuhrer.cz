Setting up HTTPS is really easy

Posted Jul 1, 2018 2018-07-01T20:55:00+02:00 by Ondřej Führer
Updated Jan 19 2023-01-19T15:57:39+01:00

Setting up HTTPS on your site is really easy, and you can get one for free. I’ve set it for this site using a great tutorial from httpsiseasy.com. That is a great series of videos from a security researcher Troy Hunt. It uses Cloudflare to easily add free SSL certificate to any site.

What is HTTPS

HTTPS is a secure version of HTTP which is the protocol used to transfer data on the web. As the HTTPS version is secure, it uses certificates to encrypt data in the transfer. That means no one can read and understand the data travelling between your device (laptop, smartphone, etc) and the server the site you’re visiting is running on.

Quick HTTPS setup

Please visit httpsiseasy.com for more detailed steps. For a quick overview, those are the steps:

  1. Create Cloudflare account
  2. Add your site to your account
  3. Change DNS settings for your site to use Cloudflare name servers
  4. Wait for Cloudflare to confirm the name server settings
  5. Enable “Always use HTTPS”
  6. Done!

Generally speaking, Cloudflare functions as a reverse proxy which means your visitors will go to their servers and Cloudflare then takes care of serving the data from original server.

As an addition it will protect your site against DDos attacks and will add caching on top of that.
